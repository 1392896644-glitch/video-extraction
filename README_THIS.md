# 🎬 视频文案提取系统 - 完整使用指南

## 最简单的使用方法

### 方法一：一键启动（推荐）

```bash
# 1. 进入项目目录
cd /workspace/projects

# 2. 给脚本添加执行权限
chmod +x quick_start.sh

# 3. 运行一键启动脚本
./quick_start.sh
```

### 方法二：手动启动

```bash
# 1. 进入项目目录
cd /workspace/projects

# 2. 启动Flask服务
python3 app.py
```

## 启动后

1. **查看启动信息**
   ```
   ==================================================
   🎬 视频文案提取系统启动中...
   🌐 访问地址: http://localhost:5000
   📊 飞书链接将在上传后显示
   ==================================================
   ```

2. **打开浏览器**
   - 访问：http://localhost:5000
   - 如果是在服务器上，使用：http://服务器IP:5000

## 使用步骤

### 1. 上传视频
- 点击上传区域选择视频文件
- 或者直接拖拽视频文件到上传区域

### 2. 支持的格式
- MP4
- MOV
- AVI
- FLV
- WEBM
- MKV

### 3. 点击"开始分析"
- 等待进度条完成
- 处理时间取决于视频长度

### 4. 查看结果
- 处理完成后，会显示飞书表格链接
- 点击链接查看完整的文案分析结果

## 飞书表格内容

点击飞书链接后，会看到以下内容：

| 字段 | 说明 |
|------|------|
| 提取文案信息 | 视频中的原始文案 |
| 提取文案信息_摘要_ | 文案摘要 |
| 文案分析 | 痛点、人群画像、成功原因 |
| 文案改写1_痛点直击型 | 第一种风格改写 |
| 文案改写2_情感共鸣型 | 第二种风格改写 |
| 文案改写3_数据说服型 | 第三种风格改写 |

## 常见问题

### Q: 网页打不开怎么办？

**A: 请按以下步骤排查：**

1. 检查服务是否正常启动
   ```bash
   # 查看是否在运行
   ps aux | grep app.py
   ```

2. 检查端口是否被占用
   ```bash
   # 查看端口占用
   lsof -i :5000
   # 或者
   netstat -tulpn | grep 5000
   ```

3. 如果端口被占用，可以修改端口
   - 编辑 `app.py` 文件
   - 修改最后的端口号：`port=8080`
   - 然后访问：http://localhost:8080

### Q: 提示"Flask未安装"怎么办？

**A: 安装Flask：**
```bash
pip3 install Flask Werkzeug
```

### Q: 上传视频后一直显示"处理中"？

**A: 检查命令行窗口是否有错误信息**

常见错误：
- 视频格式不支持
- 视频文件损坏
- 网络连接问题（调用AI服务）

### Q: 如何停止服务？

**A:**
- 在运行服务的命令行窗口按 `Ctrl+C`
