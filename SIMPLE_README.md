# 简洁版视频文案提取系统

## 快速启动

```bash
# 启动Web服务
python3 app.py
```

## 访问地址

浏览器打开：`http://localhost:5000`

## 使用说明

1. 点击或拖拽视频文件到上传区域
2. 点击"开始分析"按钮
3. 等待处理完成
4. 点击"查看飞书表格"链接查看结果

## 支持的格式

- MP4、MOV、AVI、FLV、WEBM、MKV
- 最大文件大小：500MB

## 功能

- ✅ 上传视频
- ✅ 自动提取文案
- ✅ 文案分析
- ✅ 生成5种改写
- ✅ 保存到飞书
- ✅ 返回飞书链接

就这么简单！
